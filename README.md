# Q Timeline Clone

A responsive Twitter/X-style home timeline clone for the Zaio solo project rubric. The app is branded as **Q**, while still including an X favicon and visible X logo as requested.

## Features Included

- Twitter/X-style three-column home layout with sidebar, timeline, composer, trends, subscribe panel, and who-to-follow panel.
- Responsive design for desktop, tablet, and mobile with a bottom mobile navigation bar.
- Working post composer with character counter and newly-created posts appearing at the top of the feed.
- Interactive like, repost, and bookmark buttons.
- Manual custom feature: dark mode toggle with `localStorage` persistence.
- Cursor-style custom feature 1: compose post modal from desktop and mobile action buttons.
- Cursor-style custom feature 2: Explore view with search and trending topics.
- Login modal from the profile/account button.
- X favicon and X logo assets included, while the app name is Q.

## Run Locally

Open `index.html` in a browser, or deploy this folder directly to Netlify.

## Netlify Deployment

1. Go to Netlify and create a new site.
2. Drag and drop the `q-timeline-clone` folder.
3. Submit the live Netlify link with your project.
